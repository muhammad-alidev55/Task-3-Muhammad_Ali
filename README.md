# Interactive Web Elements

A modern and beginner-friendly JavaScript project that demonstrates **DOM Manipulation** by building multiple interactive UI components using **HTML5, CSS3 and JavaScript**.

This project is designed to help beginners understand how JavaScript interacts with HTML elements, updates content dynamically, responds to user events, and modifies CSS classes without reloading the webpage.

---

# Features

-  Dark Mode Toggle
-  Interactive Counter
-  Live Character Counter
-  FAQ Show/Hide Toggle
-  Real-Time DOM Updates
-  Responsive Layout
-  Clean Modern UI
-  Lightweight & Fast
-  Pure HTML, CSS & JavaScript
-  Beginner-Friendly Code Structure

# Components Included

## 1. Dark Mode Toggle

Switch between Light Mode and Dark Mode using JavaScript by adding and removing CSS classes.

---

## 2. Counter Widget

Increase, decrease, and reset the counter dynamically.

### Features

- Increment Button
- Decrement Button
- Reset Button
- Live Number Update

---

## 3. Live Character Counter

Displays the number of characters while the user types inside the textarea.

### Features

- Real-time Updates
- Instant Character Count

---

## 4. FAQ Show / Hide Toggle

Expand or collapse an FAQ answer by clicking the question.

### Features

- Toggle Answer Visibility
- Dynamic "+" / "-" Icon
- Smooth User Interaction
